# DayPilot AI

A responsive personal routine agent app for desktop and mobile browsers.

## Open the app

Open `index.html` in a browser.

## Included

- Assistant panel for planning, task creation, priority summaries, and routine suggestions
- Task board with to-do, doing, and done states
- Smart schedule generated from open tasks and workday settings
- Quick notes and reminders
- Routine tracker
- Local browser storage
- JSON export
- Responsive desktop, tablet, and mobile layouts

## AI connection

The current version includes a local planning agent so the app works without setup. The settings screen has a provider/API key area for a later backend connection to OpenAI or another model provider.
